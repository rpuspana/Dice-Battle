game rules

Winner

The first player with a player score greater or equal than 100.

Game board details

Player  

The player name that has the dot next to it, is the active player, begining a new game round.
He can use the "roll dice" button, "end turn button", to manipulate the dice and increase his score, or the other two buttons to restart the game or reach the game's main menu.

Player score

The first sum shown under the player's name.
It represents the sum of all round scores gained by the player throughout the game.
At the end of each round this score is compared to 100, to see which player won.

Round score

The sum in the red rectangle that appears under the player's score.
Each time a user clicks on the "roll dice" button the sum of both dice is added to the existing round score (which will be 0 at first).
To add the round score to the total score, the player must end his turn by clicking the "end turn" button.
Keep in mind that different numbers shown on each dice will affect the current player's round score.
Visit the last section to lean more about the special dice rolls and their effects on the player's score and turn.

NEW GAME

Play a new game. Player score is set to 0, round score is set to 0, Player 1 will roll the dice first.

ROLL DICE

Current player rolls the dice.

END TURN

Current player ends turn. His round score gets added to his player score.

Special dice roll effect

Both dice show 6

Player score increases by 20, round score is 0, game control passed to the next player.

One of the dice shows 1, the other shows 2, 3, 4, 5 or 6

Player score remains the same, round score is set to 0, game control passed to the next player.

Both dice show 1

Player score is set to 0, round score is set to 0, game control passed to the next player.
